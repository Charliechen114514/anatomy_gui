# v0.2.0 — 2025-05-15

### Changed
- 文档站点从 MkDocs Material 迁移至 VitePress

  - 新增 C++ 模板语法转义插件（vite + markdown-it 双层），防止 `<T>` 被 Vue 编译器误解析
  - 侧边栏按 15 个主题分组自动生成
  - 首页改为 VitePress home layout，带 hero、feature cards、学习路线组件（HomeRoadmap.vue）
  - 新增 `native_win32/index.md` 章节着陆页
  - CI/CD 工作流从 Python 环境切换到 Node.js 22 + pnpm 10
  - `rc` 语言别名映射到 `c`，消除构建警告

- 移除 MkDocs 相关文件：`mkdocs.yml`、`requirements.txt`、`overrides/`、`.pages`、`tags.md`
- 移除根目录 `CHANGELOG.md`，改用 `changelog/` 目录管理版本记录
