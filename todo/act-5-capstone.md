## 应用规格：跨平台 Markdown 编辑器
- 左侧编辑区 + 右侧实时预览（500ms 防抖刷新）
- 文件操作：新建/打开/保存/另存为/最近文件列表（MRU）
- 工具栏 + 菜单栏 + 状态栏（字数/行号/编码/光标位置）
- Markdown 语法高亮
- 系统主题跟随（深色/浅色自动切换）
